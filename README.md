# E-Commerce Web App

A **responsive e-commerce website** built with **React, Vite, Tailwind CSS, Express, and Node.js**.  
Features product listings, shopping cart, checkout, and user authentication.

## Features
- Browse and search products
- Add products to cart and checkout
- User authentication and profile management
- Responsive design for mobile and desktop
- Backend API with Express and Node.js
